TPT Consultancy & Training

This project involved the design and development of a new website for TPT Consultancy & Training.

Due to the fact that client needed to display a catalogue of their offerings on the website, the challenge was to show this information in a clear and presentable manner.

The website runs on Wordpress and the theme was developed from scratch. This allowed me to gain a greater understanding of Wordpress and PHP, along with HTML, CSS3 (with SCSS) and JS.

Warcraft Character Tracker

The goal of this project is to create a character tracker for World of Warcraft that performs similar functions to the official Armory website, as well as adding additional features.

This project is being developed with Vue.js and was chosen as a way of learning the framework. It also allowed me to gain experience in utilising a public API in order to collect and present information.

The project is currently on-going and is currently fully functional for Alliance Mage and Druid characters.

(To demo the app in it's current state, the following options can be selected: Region - EU, Realm - Ravencrest, Character - Destie)
